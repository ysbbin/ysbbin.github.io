Python YOLOv5m YOLOv8m OpenAI API Streamlit Roboflow

개요

31종 한국 식재료를 YOLO 모델로 자동 인식하고, OpenAI API 챗봇이 인식된 재료 기반 맞춤 레시피를 추천하는 AI 시스템입니다.

역할

팀장 | 프로젝트 설계, 데이터 수집, 모델 비교 실험

진행 과정

  • 데이터 구축 — 수천 장의 식재료 이미지를 직접 수집하고 labelImg로 라벨링하여 자체 데이터셋 구축
  • 전략 전환 — 초기 정확도 30% 미만 확인 후, 자체 데이터는 테스트용으로 전환하고 Roboflow 기반 31클래스 외부 데이터셋을 학습에 활용

  • 서비스 구현 — Streamlit 웹 UI, 자체 레시피 DB 및 실시간 크롤링, OpenAI API 챗봇 연동

데이터 수집 → 모델링 → 최적화 → 서비스 배포까지 AI 프로젝트 전체 파이프라인을 성공적으로 완수했습니다. 최종 모델 YOLOv8m은 Precision 93.75%, mAP50 85.95%를 달성했습니다.

객체 탐지 / 강건성 분석

이미지 열화 환경에서의 객체 탐지 강건성 분석 및 개선

Python PyTorch YOLOv8m RT-DETR-L Faster R-CNN U-Net Ultralytics

개요

드론 항공 영상(VisDrone-DET)에서 이미지 열화(노이즈, 블러, 저해상도)가 객체 탐지 성능에 미치는 영향을 분석하고, 증강 학습과 U-Net 이미지 복원 전략으로 강건성을 개선하는 연구 프로젝트입니다.

진행 과정

  • Phase 1 — 베이스라인 평가 — Faster R-CNN, RT-DETR-L, YOLOv8m 3종 모델을 정상 데이터로 학습 후, 노이즈·블러·저해상도 열화 조건에서 성능 저하를 측정 (최대 -46.1%, Faster R-CNN 블러)
  • Phase 2 — 증강 학습 — 학습 데이터에 50% 열화 증강을 적용하여 재학습, 모든 모델에서 강건성 향상 확인 (YOLOv8m 블러 mAP +0.175 개선)
  • Phase 3 — U-Net 복원 — 경량 U-Net(3.70M 파라미터)을 학습하여 열화 이미지를 복원 후 탐지 수행, 블러 조건에서 우수한 성능 확인 (PSNR 34.03dB, SSIM 0.947)
  • 전략 비교 — 36개 실험 조합(3전략 × 3모델 × 4조건)을 체계적으로 비교하여 조건별 최적 전략 도출

핵심 결과

  • 범용 강건성 — YOLOv8m + 증강 학습이 평균 성능 저하율 -4.7%로 가장 안정적
  • 블러 특화 — U-Net 복원 전처리가 증강 학습보다 우수 (YOLOv8m 블러 mAP 0.640)
  • 복원 한계 — 노이즈 조건에서 U-Net 과도한 평활화로 오히려 성능 급감 (-65.2%)
  • 아키텍처 특성 — Faster R-CNN은 2단계 파이프라인 특성상 블러에 가장 취약, RT-DETR-L은 글로벌 어텐션으로 균형 잡힌 강건성 확보

성과

주식 자동 리서치 에이전트

Python Flask Gemini 2.5 Flash Google Search Gmail SMTP Windows Scheduler

개요

Gemini 2.5 Flash와 Google Search를 활용해 관심 종목·산업의 최신 뉴스와 시장 동향을 매일 자동 분석하고, 사용자별 맞춤 HTML 리포트를 Gmail로 발송하는 웹 기반 자동화 시스템입니다. 터미널 없이 브라우저만으로 모든 설정과 관리가 가능하며, 완전 무료로 운영할 수 있습니다.

시스템 구조

  • 웹 대시보드 (app.py / Flask) — API 키·Gmail 인증·관심 종목 설정, 다중 사용자 구독 관리, 즉시 발송 테스트 기능 제공. 설정은 JSON 파일로 영속 저장
  • AI 리서치 엔진 (scheduler.py) — 공통 시장 뉴스 1회 생성 후 사용자별 관심 자산을 중복 없이 캐싱하여 불필요한 API 호출 최소화
  • 자동화 (setup_scheduler.bat) — PowerShell로 Windows 작업 스케줄러에 일일 작업 등록, 웹 서버 종료 후에도 매일 지정 시간에 자동 실행

AI 분석 로직

  • 모델 — Gemini 2.5 Flash, Temperature 0.3 (정확성 중심), Google Search 도구 활성화
  • 종목 리서치 — 요약·시장해석·현재위치·매력도·리스크 5개 섹션으로 분석
  • 산업 리서치 — 요약·수급·사이클·수혜주·매력도·리스크 6개 섹션으로 분석
  • 타임프레임 분석 — 자산별 단기·중기·장기 전망을 각각 생성
  • 공통 뉴스 요약 — 당일 글로벌·한국 시장의 핵심 요인 3가지 추출

이메일 리포트 구조

  • 포트폴리오 요약 — 보유 자산별 1줄 해석 (10단어 이내)
  • 포트폴리오 리스크 — 공통 위험요소 1~2개 요약

  • 타임프레임 관점 — 단기·중기·장기 전망 통합 섹션
